Простой автономный менеджер проектов для ваших любимых проектов.

Крошечный менеджер это первое простое приложение в автономном режиме, которое помогает вам управлять любимые проекты. Наряду с управлением проектами он позволяет управлять повседневными делами. используя todos и простое приложение блокнота, все в одном месте.

В комплекте с основными функциями он позволяет быстро и легко управлять ваши проекты.

Функции

Несколько вещей, которые вы можете сделать с Крошечный менеджер:

  • Добавлять, редактировать, сохранять заметки
  • Добавить, изменить, удалить Todos
  • Управление любимыми проектами
  • Статус завершения проекта
  • Фильтрация и сортировка
  • Проблемы с установкой приоритета
  • Темный режим
  • Поддержка локализации
  • Автономный режим
  • Автономное локальное хранилище

Конфиденциальность и использование

Приложение использует локальное хранилище для сохранения вашей работы. data и не хранит никаких данных Крошечного менеджера на каком-либо сервере. Приложение использует аналитику Google, чтобы получить обзор использование приложения.
